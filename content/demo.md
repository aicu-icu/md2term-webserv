# 🎉 欢迎使用 Markdown Stream Server

这是一个**流式输出**的演示文件，支持多种Markdown语法和Emoji 😀

---

## ✨ 特性

- 🎨 彩色输出
- 📝 Markdown 渲染
- ⚡️ 可配置速度
- 🖥️ 终端友好

## 📌 代码示例

```javascript
function hello() {
  console.log("Hello, World! 🌍");
}

hello();
```

```python
def greet(name):
    return f"Hello, {name}! 👋"

print(greet("World"))
```

```rust
fn main() {
    println!("Hello, {}!", "World 🦀");
}
```
